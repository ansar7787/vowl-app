# 🛠️ Vowl Automation & Data Pipeline

This directory contains the custom automation scripts used to build, purify, and maintain the **Vowl Diamond Standard Curriculum**.

## 🚀 Purpose
Vowl is designed for massive scale (60,000+ interactive quests). Managing this volume of pedagogical data manually is impossible. These scripts represent the **Automation Engine** behind the app, handling:

- **Linguistic Purification**: Automated sweeps for 100% grammar accuracy and UTF-8 stability.
- **Contextual Mapping**: Intelligent icon and emoji assignment based on word semantics.
- **Pedagogical Enrichment**: Generating synonym-based hints and prefix-aware explanations.
- **Integrity Auditing**: Multi-pass verification to ensure zero duplicate IDs and perfect JSON structures.

## 📂 Core Scripts
- **`vocabulary_purifier.js`**: The primary engine for curriculum normalization.
- **`curriculum_audit.dart`**: A high-performance Dart script for verifying cross-module data integrity.
- **`generate_accent_shadowing.js`**: Complex logic for mapping phonetics to interactive shadowing quests.
